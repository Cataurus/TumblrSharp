GetUserLikesAsync(Int32, Int32)

Asynchronously retrieves the current user's likes.

Declaration
public Task<Likes> GetUserLikesAsync(int startIndex = 0, int count = 20)
Parameters
Type Name Description
System.Int32 startIndex

The offset at which to start retrieving the likes. Use 0 to start retrieving from the latest like.
System.Int32 count

The number of likes to retrieve. Must be between 1 and 20.
Returns
Type Description
System.Threading.Tasks.Task<Likes>

A System.Threading.Tasks.Task<TResult> that can be used to track the operation. If the task succeeds, the System.Threading.Tasks.Task`1.Result will carry a Likes instance. Otherwise System.Threading.Tasks.Task.Exception will carry a TumblrException representing the error occurred during the call.
Remarks

See: http://www.tumblr.com/docs/en/api/v2#m-ug-likes

  • count is less than 1 or greater than 20.

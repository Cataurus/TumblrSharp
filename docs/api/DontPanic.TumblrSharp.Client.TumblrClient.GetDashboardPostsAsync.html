GetDashboardPostsAsync(Int64, Int64, Int32, PostType, Boolean, Boolean)

Asynchronously retrieves posts from the current user's dashboard.

Declaration
public Task<BasePost[]> GetDashboardPostsAsync(long sinceId = 0L, long startIndex = 0L, int count = 20, PostType type = PostType.All, bool includeReblogInfo = false, bool includeNotesInfo = false)
Parameters
Type Name Description
System.Int64 sinceId

Return posts that have appeared after the specified ID. Use this parameter to page through the results: first get a set of posts, and then get posts since the last ID of the previous set.
System.Int64 startIndex

The post number to start at.
System.Int32 count

The number of posts to return.
PostType type

The PostType to return.
System.Boolean includeReblogInfo

Whether or not the response should include reblog info.
System.Boolean includeNotesInfo

Whether or not the response should include notes info.
Returns
Type Description
System.Threading.Tasks.Task<BasePost[]>

A Task<BasePost[]> that can be used to track the operation. If the task succeeds, the Task<BasePost[]>.Result"/> will carry an array of BasePost. Otherwise System.Exception will carry a TumblrException representing the error occurred during the call.
Exceptions
Type Condition
System.ObjectDisposedException

The object has been disposed.
System.InvalidOperationException

This TumblrClient instance does not have an OAuth token specified.
System.ArgumentOutOfRangeException

  • startIndex is less than 0.
  • count is less than 1 or greater than 20.

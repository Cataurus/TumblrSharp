GetQueuedPostsAsync(String, Int64, Int32, PostFilter)

Asynchronously returns posts in the current user's queue.

Declaration
public Task<BasePost[]> GetQueuedPostsAsync(string blogName, long startIndex = 0L, int count = 20, PostFilter filter = PostFilter.Html)
Parameters
Type Name Description
System.String blogName

The name of the blog for which to retrieve queued posts.
System.Int64 startIndex

The offset at which to start retrieving the posts. Use 0 to start retrieving from the latest post.
System.Int32 count

The number of posts to retrieve. Must be between 1 and 20.
PostFilter filter

A PostFilter to apply.
Returns
Type Description
System.Threading.Tasks.Task<BasePost[]>

A Task<BasePost[]> that can be used to track the operation. If the task succeeds, the Task<BasePost[]>.Result will carry an array of posts. Otherwise System.Threading.Tasks.Task.Exception will carry a TumblrException representing the error occurred during the call.
Remarks

See: http://www.tumblr.com/docs/en/api/v2#blog-queue

Exceptions
Type Condition
System.ObjectDisposedException

The object has been disposed.
System.ArgumentNullException

blogName is null.
System.ArgumentException

blogName is empty.
System.ArgumentOutOfRangeException
  • startIndex is less than 0.
  • count is less than 1 or greater than 20.
System.InvalidOperationException

This TumblrClient instance does not have an OAuth token specified.

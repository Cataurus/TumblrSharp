The name of the method to call. The method url will be automatically built.

The OAuth Token to use for the call. Can be null if the method does not require OAuth.

The required System.Net.Http.HttpMethod for the Tumblr API call. Only System.Net.Http.HttpMethod.Get and System.Net.Http.HttpMethod.Post are supported.

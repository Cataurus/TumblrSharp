GetFollowersAsync(String, Int32, Int32)

Asynchronously retrieves a blog's followers.

Declaration
public Task<Followers> GetFollowersAsync(string blogName, int startIndex = 0, int count = 20)
Parameters
Type Name Description
System.String blogName

The name of the blog.
System.Int32 startIndex

The offset at which to start retrieving the followers. Use 0 to start retrieving from the latest follower.
System.Int32 count

The number of followers to retrieve. Must be between 1 and 20.
Returns
Type Description
System.Threading.Tasks.Task<Followers>

A System.Threading.Tasks.Task<TResult> that can be used to track the operation. If the task succeeds, the System.Threading.Tasks.Task`1.Result will carry a Followers instance. Otherwise System.Threading.Tasks.Task.Exception will carry a TumblrException A Followers instance.
Remarks

See: http://www.tumblr.com/docs/en/api/v2#blog-followers

Exceptions
Type Condition
System.ObjectDisposedException

The object has been disposed.
System.ArgumentNullException

blogName is null.
System.ArgumentException

blogName is empty.
System.ArgumentOutOfRangeException
  • startIndex is less than 0.
  • count is less than 1 or greater than 20.

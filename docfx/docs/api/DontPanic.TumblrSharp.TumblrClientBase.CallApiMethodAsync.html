CallApiMethodAsync<TResponse, TResult>(ApiMethod, Func<TResponse, TResult>, CancellationToken, IEnumerable<JsonConverter>)

Asynchronously invokes a method on the Tumblr API, and performs a projection on the response before returning the result.

Declaration
public Task<TResult> CallApiMethodAsync<TResponse, TResult>(ApiMethod method, Func<TResponse, TResult> projection, CancellationToken cancellationToken, IEnumerable<JsonConverter> converters = null)
    where TResponse : class where TResult : class
Parameters
Type Name Description
ApiMethod method

The ApiMethod to invoke.
System.Func<TResponse, TResult> projection

The projection function that transforms TResponse into TResult.
System.Threading.CancellationToken cancellationToken

A System.Threading.CancellationToken that can be used to cancel the operation.
System.Collections.Generic.IEnumerable<Newtonsoft.Json.JsonConverter> converters

An optional list of JSON converters that will be used while deserializing the response from the Tumblr API.
Returns
Type Description
System.Threading.Tasks.Task<TResult>

A System.Threading.Tasks.Task<TResult> that can be used to track the operation. If the task succeeds, the System.Threading.Tasks.Task`1.Result will carry a TResult instance. Otherwise System.Threading.Tasks.Task.Exception will carry a TumblrException representing the error occurred during the call.
Type Parameters
Name Description
TResponse

The type of the response received from the API. This must be a type that can be deserialized from the response JSON.
TResult

The actual type that is the result of the method.
Exceptions
Type Condition
System.ObjectDisposedException

The object has been disposed.
System.ArgumentNullException
  • method is null.
  • projection is null.
